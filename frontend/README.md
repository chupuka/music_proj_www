Фронтенд для музыкального стримингового приложения.

```
frontend/
├── index.html      # Главная страница
├── css/
│   └── styles.css # Стили приложения
├── js/
│   ├── api.js     # API клиент для работы с бэкендом
│   └── app.js     # Основная логика приложения
└── README.md      # Документация
```

## Запуск

### Вариант 1: Простой локальный сервер

1. Установите простой HTTP сервер (если нет):
   ```bash
   npm install -g http-server
   ```

2. Перейдите в директорию frontend:
   ```bash
   cd frontend
   ```

3. Запустите сервер:
   ```bash
   http-server -p 3000
   ```

4. Откройте браузер: `http://localhost:3000`

### Вариант 2: Python HTTP сервер

```bash
cd frontend
python -m http.server 3000
```

Затем откройте: `http://localhost:3000`

### Вариант 3: Live Server в VS Code

1. Установите расширение "Live Server" в VS Code
2. Правый клик на `index.html` → "Open with Live Server"

## API Интеграция

Приложение подключается к API Gateway по адресу: `http://localhost:8080`

### Endpoints используемые:
- `GET /api/users` - Получение списка пользователей
- `POST /api/users` - Регистрация нового пользователя
- `GET /api/tracks` - Получение списка треков
- `GET /api/artists` - Получение списка исполнителей
- `GET /api/playlists/user/{userId}` - Получение плейлистов пользователя
- `GET /api/favorite-tracks/user/{userId}` - Получение избранных треков
- `POST /api/favorite-tracks` - Добавление трека в избранное
- `DELETE /api/favorite-tracks/user/{userId}/track/{trackId}` - Удаление из избранного

## Настройка API URL

Если ваш API Gateway запущен на другом порту, измените в `js/api.js`:

```javascript
const API_BASE_URL = 'http://localhost:8080'; // Ваш порт
```
